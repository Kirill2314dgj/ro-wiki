How to Create a Roblox Game Pass

Use this article as a reusable template. Replace headings, text blocks, and image placeholders to publish new tutorials quickly.

Overview

This guide explains how to create a game pass, set a price, grab the pass ID, and connect ownership checks to in-game features.

Requirements

  • A published Roblox experience.
  • Access to Creator Dashboard and Studio.
  • Basic Lua scripting knowledge.

Step-by-step

  1. Create Pass: In Creator Dashboard, open Monetization → Passes and upload a thumbnail.
  2. Set Price: Enable item for sale and choose your Robux amount.
  3. Copy ID: Open the pass page URL and copy the numeric ID.
  4. Add Script: Use MarketplaceService:UserOwnsGamePassAsync() to unlock benefits.
  5. Test: Verify in Studio and in a live server with test accounts.

Code snippet

local MarketplaceService = game:GetService("MarketplaceService")
local PASS_ID = 123456789

local function playerOwnsPass(player)
  local success, ownsPass = pcall(function()
    return MarketplaceService:UserOwnsGamePassAsync(player.UserId, PASS_ID)
  end)
  return success and ownsPass
end

FAQ

Why does ownership check fail?

Usually because of wrong pass ID, unpublished game, or temporary API errors. Wrap checks with pcall.
