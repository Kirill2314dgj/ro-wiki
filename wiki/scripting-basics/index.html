Scripting Basics

Understand how scripts react to player actions.

Start with these concepts

  • Variables store values.
  • Functions run reusable logic.
  • Events trigger scripts from actions.

Simple example

local part = script.Parent
part.Touched:Connect(function(hit)
  print(hit.Name .. " touched the part")
end)
