# RO-Wiki

SEO-ready static website template for Roblox tutorials.

## Pages
- `/` home page
- `/tutorials/` tutorials hub
- `/tutorials/how-to-gamepass/` full article template
- `/tutorials/tower-building/` example article
- `/tutorials/scripting-basics/` example article
- `/policy/` policy page
- `/404.html` custom not-found page

## Features
- Dark theme by default + light theme toggle.
- Reusable top navigation and footer blocks.
- OpenGraph, canonical URLs, robots directives, and JSON-LD metadata.
- `robots.txt` + `sitemap.xml` for indexing.

## Quick editing workflow
1. Copy `tutorials/how-to-gamepass/index.html` into a new folder under `tutorials/`.
2. Replace title, description, canonical URL, and section content.
3. Add your screenshots to `assets/img/` and update `img` paths + `alt` text.
4. Link the new article from `tutorials/index.html`.
